== README

Readme for {Sheep Tracking System}[http://wolfbane.heroku.com]

Requirements for running the application locally
* A POSIX-compatible / UNIX-based Operating System
* Ruby 2.0.0
* PostgreSQL 9.3
* Rails 4.0.0
* All gems as specified by Gemfile

Guide for installation and setup: http://edgeguides.rubyonrails.org/getting_started.html
